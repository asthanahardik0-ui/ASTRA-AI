# 🚀 ASTRA AI

## Artificial Student Tracking and Research Assistant

ASTRA AI is an intelligent student assistant developed using **Python**, **Streamlit**, and **Google Gemini AI**.

It helps students study smarter by generating notes, solving doubts, creating quizzes, explaining code, chatting with PDFs, preparing study plans, and tracking learning progress.

---

# ✨ Features

- 🤖 AI Doubt Solver
- 📝 AI Notes Generator
- 📚 Study Coach
- ❓ Quiz & Flashcards Generator
- 💻 Code Explainer
- 📄 PDF Chat
- 📅 Timetable Generator
- 📊 Productivity Dashboard
- 🏆 XP & Achievement System
- 👤 Multi-User Login System
- ⚙️ Settings Page
